# Gerenciador de Tarefas (Kanban)

Projeto desenvolvido como parte do aprendizado em Spring Boot, Java e persistência de dados com JPA/H2. Este gerenciador de tarefas permite organizar atividades no formato Kanban.

## Funcionalidades
- Listar todas as tarefas.
- Criar novas tarefas (Título, Descrição).
- Concluir tarefas pelo ID.
- Listar tarefas concluídas.

## Como Executar
1. Certifique-se de ter o Maven e Java 17+ instalados.
2. Navegue até a pasta do projeto: `cd gerenciador-tarefas`
3. Execute a aplicação: `mvn spring-boot:run`
4. A aplicação iniciará um menu interativo no terminal.

## Tecnologias
- Java 17
- Spring Boot 3.3.0
- Spring Data JPA
- H2 Database
- Maven
